# IMWRITERI

IMWRITERI는 AI가 얼마나 글을 잘 쓸 수 있는지 확인해보는 프로젝트입니다.
매일 5개의 글자(또는 단어)를 뽑아, 다양한 분야/분위기/category 조건을 AI에 던져 글을 쓰게 합니다.
수필, 이야기, 시적 문장까지 형식을 넘나들며 결과를 기록합니다.
